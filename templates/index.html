Student API

Create Student Profile

Create a new student profile.

                    POST /student/create_student_profile
                    Parameters:
                    - user_id
                    - username
                    - password
                    - user_class
                    - status_title
                    - status_description
                    - about
                    - phone
                    - email
                    - address
                    - parents
                    - image (File)

Get Student Profile

Retrieve a student's profile by user_id.

                    GET /student/get_user/{user_id}

Update Student Profile

Update a student's profile data by user_id.

                    PUT /student/update_student_profile/{user_id}
                    Parameters (form data):
                    - username
                    - password
                    - user_class
                    - status_title
                    - status_description
                    - about
                    - phone
                    - email
                    - address
                    - parents
                    - image (File)

Search Students

Search for students by username, user_id, phone, or email.

                    GET /student/search/{query}
